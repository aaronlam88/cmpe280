# Requirements
This program is tested under the following conditions. If your environment is different, please use Google for help with errors
```
node --version ## v10.5.0
npm --version ## 6.4.0
```
# For Normal User
## Install
```
npm install
```
## Run
```
npm start
```

# For Developer
## Install
```
npm install
npm run devinstall
```

## To run with hot reload
```
npm run devstart
```

## To run without hot reload
```
npm start
```
